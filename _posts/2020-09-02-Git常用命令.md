- **git提交至github**

  git status

  git add .

  git commit -m 'message'

  git push

- **常用命令**

  git clone : 从github克隆至本地

  git add . : 添加所有文件

  git commit : 提交

  git push : 推送至github

  git status : 查看状态

  git log ： 查看日志

  git log --pretty=oneline : 每次提交显示为一行

  git checkout . : 撤销到上一次提交

  > git checkout be017f : 回退到任意版本
  >
  > git checkout master : 返回到master分支
